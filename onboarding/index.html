One quick setup to enable free AI summaries on YouTube.

Get Your Free API Key

To get started, we need to connect to a free AI service. This requires an "API key."

What's an API key? Think of it like a special password that lets this extension securely access the AI. It's a standard and safe way for apps to connect to services.

We'll guide you through getting a free one in just a few clicks.
